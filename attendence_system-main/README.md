# 🎓 Face Recognition Attendance System

This project is a real-time face recognition attendance system built using Python and OpenCV. It detects and recognizes faces from webcam input and records attendance based on known images.

---

## 📁 Project Structure

.
├── attendance_system.py
├── requirements.txt
├── README.md
└── images/
├── person1.jpeg
├── person2.jpeg
├── person3.jpeg
└── person4.jpeg


---

## 🚀 Features

- Real-time face detection using webcam
- Recognizes known faces from the `images/` folder
- Logs attendance in a CSV file
- Avoids marking the same person twice in one run
